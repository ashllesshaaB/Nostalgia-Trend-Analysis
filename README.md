# Nostalgia Trend Analysis Dashboard

This project analyzes viral content trends across social media platforms using clustering and statistical methods.

## Features
- Viral spike detection using engagement metrics
- K-Means clustering for content grouping
- Chi-square statistical testing for platform analysis
- Interactive dashboard using Streamlit
- Power BI dashboard for business insights

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Power BI

## How to Run
1. Install dependencies
2. Run:
   python -m streamlit run report/app.py
